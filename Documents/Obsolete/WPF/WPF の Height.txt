WPF の Height

// TextBox
Visual Studio のデザイナーでは、ウィンドウの端のコントロールと境界線との間が10ピクセルに、
    隣接するコントロールの間が5ピクセルになる
その5という数字を引き継ぎ、TextBox に上下5ピクセルのパディングを与え、
    文字の高さと行高が一致する MS UI Gothic の 12px で文字列を設定すれば、高さは24ピクセルになる
そこから上下パディングをなくし、VerticalContentAlignment を Center にしたものが Nekote の TextBox
24という数字は、フォントサイズのちょうど倍というキリの良さもあり、UI 全体で整合させやすい
