WPF の TextBox

// 基本
x:Name を設定
Text を設定または消す

// 一行
必要に応じて StackPanel または DockPanel に入れて可変幅に
Height="24"
// 「WPF の Height.txt」を参照
Padding="3,0"
// 「WPF の Padding.txt」を参照
VerticalContentAlignment="Center"
TextWrapping を消す

// StackPanel
// こちらには高さを設定しない
Orientation="Horizontal"

// DockPanel
// LastChildFill="True" はなくてよい
// デフォルトでそうなっている

// 複数行
Padding="8"
// 「WPF の Padding.txt」を参照
VerticalContentAlignment があれば消す
FontSize="15"
// 「WPF のフォント.txt」を参照
TextWrapping="Wrap"
VerticalScrollBarVisibility="Auto"
// AcceptsTab="True" は不要
// ソフトタブが主流の時代だし、
//     タブキーで移動できないと不便
AcceptsReturn="True"

// 読み取り専用
Background="#F0F0F0"
// 入力できないことを分かりやすく
IsTabStop="False"
IsReadOnly="True"
AcceptsReturn など不要なものがあれば消す
