WPF のフォント

// ファミリー
Windows 10/11 ではデフォルトが Yu Gothic UI
限られたスペースで多くを表示でき、よく調整されている
ただ、老眼だと見にくいかもしれないし、文章の入力に適するものでない
そのため、今でも良くできていると思う Meiryo に
欧米でも使えるし、Segoe UI などのモダンな UI フォントとの相性も良い
「メイリオ」と指定しないのは、これはあくまでローカル名だから

// サイズ
デフォルトは12ピクセル
そのままでよいが、コントロールによっては継承されないかもしれないし、
    Windows の設定などに影響されて表示が崩れる可能性もあるので明示的に

// 複数行の TextBox
ファミリーはそのままでサイズを25％増しの15ピクセルにする
たいてい一行で十分で、複数行にするところにはそれなりの量の文章を書く
そこに集中して働くのだから少し大きめに

// StatusBar
Window で指定したフォントファミリーが反映されないようだ
他にもそういうコントロールが見付かればファミリーとサイズの両方を指定
